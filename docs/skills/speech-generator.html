1 מה צריך לפני שמתחילים?

חשבון ElevenLabs

הסקיל משתמש בשירות ElevenLabs - אחד משירותי ה-TTS הטובים בעולם. הם מציעים קולות שנשמעים מאוד אנושיים.

💰 כמה זה עולה?

יש תוכנית חינמית! מקבלים 10,000 תווים בחודש בחינם. זה מספיק ל:

  • כ-10 דקות של אודיו
  • כ-50 הודעות קוליות קצרות
  • כ-5-10 פודקאסטים קצרים

אם צריכים יותר, התוכניות מתחילות מ-$5 לחודש.

2 יצירת חשבון ElevenLabs

  1. היכנסו לאתר ElevenLabs

    לחצו על הכפתור כדי להיכנס:

    🌐 היכנסו ל-ElevenLabs
  2. לחצו על "Sign Up"

    בפינה הימנית העליונה. אפשר להירשם עם Google או אימייל.

  3. אשרו את האימייל

    אם נרשמתם עם אימייל - לחצו על הקישור שתקבלו.

  4. השלימו את ההרשמה

    ענו על כמה שאלות קצרות (מה אתם מתכננים ליצור וכו').

3 קבלת מפתח API

עכשיו נקבל את המפתח שקלוד צריך:

  1. היכנסו להגדרות

    לחצו על התמונה שלכם (או האות הראשונה של השם) בפינה הימנית העליונה, ובחרו Profile + API key.

  2. העתיקו את ה-API Key

    בחלק של "API Key" לחצו על הכפתור להצגה (👁️) ואז על כפתור ההעתקה.

    המפתח נראה כמו: sk_abc123def456...

🔒

שמרו על המפתח! זה כמו סיסמה. אל תשתפו אותו ואל תעלו אותו לאינטרנט.

4 בחירת קול (אופציונלי)

ElevenLabs מציעים מאות קולות שונים. אפשר להשתמש בקול מוכן או אפילו לשכפל את הקול שלכם!

איך למצוא קול שמתאים לכם:

  1. היכנסו ל-Voice Library

    בתפריט השמאלי לחצו על Voices ואז Voice Library.

  2. סננו לפי שפה

    לחצו על Filters ובחרו Hebrew (עברית) אם אתם רוצים קול שתומך בעברית.

  3. האזינו ובחרו

    לחצו על כפתור ההשמעה ליד כל קול. כשמצאתם קול שאהבתם - לחצו עליו.

  4. העתיקו את ה-Voice ID

    בדף הקול, ליד "Voice ID" יש כפתור העתקה. לחצו עליו.

    ה-ID נראה כמו: dV1ee5wE1Ag5NSL6L2Z9

🎤 קולות מומלצים לעברית

חפשו ב-Voice Library קולות עם התגית "Hebrew". יש גם קולות רב-לשוניים שעובדים טוב עם עברית.

💡

טיפ: אם אתם לא בוחרים קול, קלוד ישתמש בקול ברירת המחדל.

⚠️ לפני שממשיכים - הגדרת פרטיות

בשלב הבא תזינו מפתח API. זה כמו סיסמה, ולכן חשוב לוודא שהגדרות הפרטיות מוגדרות נכון בקלוד קוד.

🔐

אם עדיין לא עשיתם את זה, עברו קודם למדריך הגדרות פרטיות וודאו שכיביתם את האפשרות לאנתרופיק להתאמן על השיחות שלכם.

5 הגדרת הסקיל במחשב

הכי קל - לבקש מקלוד קוד לעשות את זה בשבילכם!

🪄 הדרך הקלה - בקשו מקלוד

פתחו את קלוד קוד ואמרו לו:

צור קובץ .env בתיקייה ~/.claude/skills/speech-generator/scripts עם התוכן הבא: ELEVENLABS_API_KEY=sk_המפתח_שהעתקתם_מהאתר DEFAULT_VOICE_ID=dV1ee5wE1Ag5NSL6L2Z9

קלוד ייצור את הקובץ במקום הנכון!

* ה-DEFAULT_VOICE_ID הוא אופציונלי. אם לא תגדירו, תצטרכו לציין קול בכל פעם.

התקנת ספריות

אחרי שהקובץ נוצר, בקשו מקלוד:

הרץ npm install בתיקייה ~/.claude/skills/speech-generator/scripts
📝

דוגמה לערכים אמיתיים:

ELEVENLABS_API_KEY=sk_abc123def456ghi789jkl012mno345 DEFAULT_VOICE_ID=dV1ee5wE1Ag5NSL6L2Z9

6 בדיקה שהכל עובד

הפעילו מחדש את קלוד קוד ונסו:

# בקשו מקלוד: צור קובץ אודיו שאומר "שלום עולם, זו בדיקה"

קלוד ייצור קובץ MP3 עם ההקלטה. אם שמעתם את הקול - מזל טוב! 🎉

דוגמאות לשימוש

# הקלטה פשוטה "צור הקלטה שאומרת: היי, מה נשמע?" # הקלטה עם רגש "צור הקלטה נלהבת שאומרת: וואו! זה מדהים!" # הודעה קולית לוואטסאפ (עם סקיל WhatsApp) "צור הקלטה ושלח אותה בוואטסאפ לדני" # הקלטה ארוכה מקובץ "צור הקלטה מהטקסט שבקובץ script.txt"

הנחיות להבעה (עובד בעברית!)

אפשר להוסיף הנחיות בסוגריים מרובעים:

[בהתלהבות] וואו! זה נהדר! [בעצב] אני מצטער לשמוע... [בשקט] זה סוד, אבל... [צחוק] הההה, זה ממש מצחיק [נשימה] אוקיי, אז... [מהר] צריך לרוץ, אין זמן!

🔧 פתרון בעיות נפוצות

שגיאת "Invalid API Key"

  • וודאו שהעתקתם את כל המפתח (מתחיל ב-sk_)
  • בדקו שאין רווחים לפני או אחרי המפתח

הקול לא נשמע טוב בעברית

  • נסו קול אחר מה-Voice Library עם תגית Hebrew
  • השתמשו בדגם eleven_multilingual_v2
