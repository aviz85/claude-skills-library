WordPress Publisher

1 Prerequisites

Before installing this skill, make sure you have:

  • Claude Code installed on your machine
  • A WordPress site with REST API enabled (enabled by default)
  • Administrator access to your WordPress site

2 Create Application Password

WordPress Application Passwords allow secure API access without exposing your main password.

  1. Log in to your WordPress Admin Dashboard
  2. Go to Users ‚Üí Profile
  3. Scroll down to Application Passwords section
  4. Enter a name like Claude Code
  5. Click Add New Application Password
  6. Copy the generated password (shown with spaces)

4 Configure Credentials

Create a .env file in the skill directory with your credentials:

# ~/.claude/skills/wordpress-publisher/.env

# Your WordPress site URL (no trailing slash)
WP_URL=https://your-site.com

# WordPress username
WP_USERNAME=your_username

# Application Password (no spaces!)
WP_APP_PASSWORD=YourAppPasswordNoSpaces

Troubleshooting

401 Unauthorized

Check that your username and application password are correct. Make sure there are no spaces in the password.

403 Forbidden

Your WordPress user might not have Editor or Administrator role. Only users with publishing permissions can use this skill.

404 Not Found

Check that your WP_URL is correct and the WordPress REST API is enabled. Some security plugins disable the REST API.

Connection Refused

Make sure your WordPress site is accessible and uses HTTPS.
