מה הסקיל הזה עושה?

הסקיל הזה מאפשר לקלוד לפרסם פוסטים באתר הוורדפרס שלכם. אתם כותבים את התוכן, קלוד יוצר טיוטה, ואחרי שאתם מאשרים - הפוסט עולה לאוויר!

משתמש ב-Application Passwords

1 מה צריך לפני שמתחילים?

  • קלוד קוד מותקן במחשב
  • אתר וורדפרס עם REST API מופעל (מופעל כברירת מחדל)
  • הרשאות מנהל או עורך באתר הוורדפרס
💡

הסקיל עובד עם כל אתר וורדפרס - גם וורדפרס.קום וגם אתרים מאוחסנים עצמאית.

2 יצירת סיסמת אפליקציה בוורדפרס

וורדפרס מאפשר ליצור "סיסמאות אפליקציה" שנותנות גישה ל-API בלי לחשוף את הסיסמה הראשית שלכם. זה הרבה יותר בטוח!

  1. היכנסו לפאנל הניהול של וורדפרס

    גלשו לכתובת האתר שלכם עם /wp-admin בסוף והתחברו.

  2. עברו להגדרות המשתמש

    בתפריט הצד לחצו על משתמשים ואז על הפרופיל שלך (או Users → Profile באנגלית).

  3. גללו למטה לחלק "סיסמאות אפליקציה"

    תמצאו שם סעיף שנקרא Application Passwords או "סיסמאות אפליקציה".

  4. תנו שם לסיסמה החדשה

    בשדה "שם סיסמת האפליקציה החדשה" כתבו שם כמו Claude Code - זה רק לזיהוי שלכם.

  5. לחצו על "הוסף סיסמת אפליקציה חדשה"

    לחצו על הכפתור Add New Application Password או "הוסף סיסמת אפליקציה חדשה".

  6. העתיקו את הסיסמה

    תקבלו סיסמה שנראית ככה: LPOo ZaEi kJLf bNmQ

    חשוב! העתיקו אותה עכשיו - לא תוכלו לראות אותה שוב!

⚠️

חשוב מאוד: כשתשתמשו בסיסמה, מחקו את כל הרווחים! הסיסמה צריכה להיות רציפה.
לדוגמה: LPOo ZaEi kJLf bNmQ הופך ל-LPOoZaEikJLfbNmQ

📌

לא רואים את האפשרות? וודאו שאתם משתמשים בחשבון עם הרשאות עורך או מנהל. משתמשים רגילים לא יכולים ליצור סיסמאות אפליקציה.

⚠️ לפני שממשיכים - הגדרת פרטיות

בשלב הבא תזינו מפתחות API. אלה כמו סיסמאות, ולכן חשוב לוודא שהגדרות הפרטיות מוגדרות נכון בקלוד קוד.

🔐

אם עדיין לא עשיתם את זה, עברו קודם למדריך הגדרות פרטיות וודאו שכיביתם את האפשרות לאנתרופיק להתאמן על השיחות שלכם.

3 הגדרת הסקיל במחשב

עכשיו נגדיר את הסקיל עם הפרטים שלכם. הכי קל - לבקש מקלוד קוד לעשות את זה בשבילכם!

🪄 הדרך הקלה - בקשו מקלוד

פתחו את קלוד קוד ואמרו לו:

צור קובץ .env בתיקייה ~/.claude/skills/wordpress-publisher עם התוכן הבא: WP_URL=https://your-site.com WP_USERNAME=שם_המשתמש_שלכם WP_APP_PASSWORD=הסיסמה_בלי_רווחים

קלוד ייצור את הקובץ במקום הנכון!

📝

דוגמה לערכים אמיתיים:

WP_URL=https://myblog.co.il WP_USERNAME=admin WP_APP_PASSWORD=LPOoZaEikJLfbNmQ
✅

קובץ ה-.env מוגן ב-gitignore, כך שהסיסמאות שלכם לא יעלו ל-GitHub.

5 בדיקה שהכל עובד

הפעילו מחדש את קלוד קוד ונסו לפרסם פוסט בדיקה:

# פתחו שיחה חדשה עם קלוד ואמרו: פרסם פוסט בדיקה בוורדפרס עם הכותרת "פוסט בדיקה" והתוכן "זהו פוסט בדיקה"

קלוד ייצור טיוטה ויתן לכם קישור לבדוק אותה. אם הכל עובד - מזל טוב! 🎉

דוגמאות לשימוש

הנה כמה דברים שתוכלו לבקש מקלוד:

# פרסום תוכן פשוט "פרסם את המאמר הזה בוורדפרס: # כותרת המאמר שלי זה התוכן של המאמר..." # פרסום מקובץ "פרסם את הקובץ ~/blog/post.md לוורדפרס" # אישור פרסום "כן, תפרסם את הטיוטה"

🔧 פתרון בעיות נפוצות

שגיאת 401 Unauthorized

  • וודאו ששם המשתמש נכון
  • וודאו שהסיסמה הועתקה בלי רווחים
  • נסו ליצור סיסמת אפליקציה חדשה

שגיאת 403 Forbidden

  • המשתמש צריך הרשאות עורך או מנהל
  • בדקו שלמשתמש יש הרשאה לפרסם פוסטים

שגיאת 404 Not Found

  • וודאו שכתובת האתר נכונה (בלי / בסוף)
  • בדקו ש-REST API מופעל (חלק מתוספי אבטחה מכבים אותו)

לא רואים "סיסמאות אפליקציה" בוורדפרס

  • וודאו שאתם מחוברים כמנהל או עורך
  • האפשרות זמינה רק אם האתר תומך ב-HTTPS
  • חלק מתוספי אבטחה מסתירים את האפשרות הזו
