# AVIZ Skills Installer Plugin

A Claude Code plugin that helps you install skills from the AVIZ Skills Library.

## Installation

### Option 1: Install from GitHub (Recommended)

```bash
# Add the marketplace
/plugin marketplace add aviz85/claude-skills-library

# Install the plugin
/plugin install aviz-skills-installer
```

### Option 2: Direct installation

```bash
claude plugin install https://github.com/aviz85/claude-skills-library
```

## Usage

After installation, just ask Claude:

- "I want to install a skill from the AVIZ library"
- "Show me available skills"
- "Help me install the whatsapp skill"

## Available Skills

Visit https://aviz.github.io/claude-skills-library/ for the full list of available skills.
